# 🌾 Agritech Platform

Agritech Platform is a full-stack web application built using Django and PostgreSQL that helps farmers access agricultural services through a single platform.

## Features

### Farmer Module

* Farmer Registration & Login
* Farmer Dashboard
* Profile Management

### Crop Support

* Crop Problem Submission
* Expert/Admin Responses
* Problem Tracking

### Consultation Module

* Consultation Booking
* Appointment Management
* Consultation History

### Machinery Module

* Machinery Listing
* Machinery Booking
* Booking History

### Services Module

* Agricultural Service Requests
* Service Tracking

### Product Module

* Crop and Seed Catalog
* Product Details
* Customer Reviews & Ratings

### E-Commerce

* Shopping Cart
* Quantity Selection
* Checkout System
* Order Management

### Dashboard Features

* My Orders
* My Consultations
* My Machinery Bookings
* My Service Requests
* My Crop Problems

## Technology Stack

* Backend: Django (Python)
* Database: PostgreSQL (Neon)
* Frontend: HTML, CSS, Bootstrap, JavaScript
* Version Control: Git & GitHub

## Installation

```bash
git clone <repository-url>
cd agritech-platform

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
